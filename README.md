# New Nikhil Khad Bhandar Website

A mobile-app like website for an agriculture business.

## Pages
1. **Home** (`index.html`): Dashboard and Welcome.
2. **Catalog** (`catalog.html`): Products grid.
3. **Contact** (`contact.html`): Map and support.

## Features
- Fixed Bottom Navigation.
- WhatsApp Integration.
- Google Maps Embed.
- Agriculture themed UI.
